변수
	
	1.하나의 값을 저장할 수 있는 메모리 공간
	2.변하는 수를 담을 수 있는 하나의 그릇과도 같음.
	3.메모리 번지에 대한 이름
	
변수 작성 규칙
	
	1.영문자, 숫자, 특수문자의 조합가능.
	2.특수문자는 달러와 언더스코어(_)만 쓸 수 있음
	3.첫번째 문제는 숫자가 나와서는 안됨.
	4.길이의 제한이 없지만 너무 길면 가독성이 떨어지니 길게 쓰진 않을 것
	5.자바 예약어(=key word,reserved word)는 사용할 수 없음.
	6.대소문자 구별해야함=abc와 Abc는 다른 변수
	
변수의 선언
	
	1.타입 변수이름
	2.타입 변수이름=초기값
	3.예제: 정수형 변수를 선언
		int x;
		int y=10;
		int z=-9;
		
데이터 타입
	
	1.정수(byte, char, short, int, long) : 0, 양수, 음수
	2.실수(float, double) : 부동소수점, 지
	3.논리(boolean) : 참과 거짓
	4.사용자(class, enum) : 정수, 실수, 논리 사용자형을 복합적으로 조합한 형태
	5.원시타입 : 기본타입(정수, 실수, 논리)
	
할당 연산자 := 

	1.변수 = 값 (값 -> 변수)
	2.데이터를 변수에 대입
	3.기존의 변수에 있던 값은 사라지고 대입되는 값이 저장
	4.예 : 변수 x를 선언하고 초기값으로 10을 할당한 후에 99를 넣으면
			기존에 있던 값인 10은 사라지고 99가 변수 x에 저장
	
		int x=10; // x<-10
		x=99;	// x<-99
		
리터럴

	1.정수 리터럴, 실수 리터럴, 문자(열) 리터럴, 논리 리터럴
	2.예
		10,-5,3.14,'a',"abc",true,false
	
상수값

	1.변하지 않는 수
	2.변하지 않는 문자 및 문자열
	3.한 번 지정되면 값을 바꿀 수 없는 변수(variable)
	
원시타입

boolean(2byte),short(2byte),char(2byte)<int(4byte)<float(4byte)<double(8byte)

원시타입의 값의 범위

	1.논리형(boolean) : true, false
	2.바이트(byte) : -128~127
	3.문자(char) : 0~65535
	4.쇼트(short) : -32768~32767
	5.정수(int) : -2147483648~2147483647
	6.실수(float) : 1.4E-45~3.402E38
	7.실수(double) : 4.9E-324~1.79E308
	
바이트타입
	
	0: 0000 0000
	1: 0000 0001
	127: 0111 1111
